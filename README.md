# Bra Fitting Photos Board

A web application for creating bra fitting photo boards. Upload or paste photos for front, 3/4, side, and back views, edit them with zoom, rotate, and fit options, then export to high-quality PNG or print-ready PDF.

## Features

- Upload multiple photos at once or individually
- Drag and drop photos
- Paste images from clipboard
- Zoom, rotate, and adjust photo positioning
- Fit photos to cover or contain the frame
- Preview the final board layout
- Export as PNG (with selectable quality)
- Export as PDF for printing
- Save and load projects as JSON

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Node.js (optional, for local development server)

### Running Locally

1. Clone or download the project files.
2. If you have Node.js installed:
   - Run `npm install` to install dependencies.
   - Run `npm start` to start a local development server.
   - Open `http://localhost:8080` in your browser.
3. Alternatively, open `index.html` directly in your browser.

### Usage

1. Open the application in your browser.
2. Click "Add 4 Photos" to upload multiple images at once, or use individual "Upload" buttons.
3. Alternatively, paste images from your clipboard or drag and drop them onto the slots.
4. Use the controls to zoom, rotate, and position each photo.
5. Click "Review Preview" to see the final layout.
6. Export as PNG or PDF using the buttons in the header.

## Project Structure

- `index.html` - Main HTML file
- `style.css` - Stylesheet
- `script.js` - JavaScript functionality
- `package.json` - Node.js project configuration
